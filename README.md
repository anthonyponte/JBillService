## Requisitos
Java 11
## Caraterísticas
* Ingresar comunicación de baja/resumen de reversiones
* Ingresar resumen diario
* Enviar documentos ingresados
* Consultar comunicaciones de baja enviados
* Consultar resumenes diarios enviados
